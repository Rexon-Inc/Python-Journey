Fundamental Data Types in Python

Basic Data Types

int, float, complex, str, bool, list, tuple, set, dict

1. Integer (int)

Integers are whole numbers. Basic mathematical operations can easily be performed.

print(type(1+2))  # int type class
print(1-2)
print(1*2)

2. Float

Floats represent decimal numbers. They use more memory than integers.

print(1/2)  # float data type = any kind of decimal output

3. Complex

Complex numbers are used in complex mathematical calculations (x+yi).

Math Functions

print(round(2.6))  # Round off to the closest integer
print(abs(-100))   # Prints absolute values/no negative numbers

Operator Precedence

Also called operator priority, it follows the rule "PEMDAS":

  • P – Parentheses
  • E – Exponentiation
  • M – Multiplication (Multiplication and division have the same precedence)
  • D – Division
  • A – Addition (Addition and subtraction have the same precedence)
  • S – Subtraction
